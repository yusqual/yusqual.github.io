392. 判断子序列

题目:

给定字符串 s 和 t ，判断 s 是否为 t 的子序列。

字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。(例如，"ace"是"abcde"的一个子序列，而"aec"不是)。

方法:

  1. 动态规划, 思路与1143.最长公共子序列差不多, 唯一的区别在递推公式.

  • dp数组以及下标含义:

    • dp[i][j]表示以下标i-1为结尾的字符串s, 和以下标j-1结尾的字符串t, 相同的子序列长度为dp[i][j]

  • 递推公式

    考虑以下两种情况:

    • s[i-1] == t[j-1], t中找到一个字符也在s中出现, dp[i][j] = dp[i-1][j-1] + 1

    • s[i-1] != t[j-1], 相当于t要删除字符再匹配, t如果把当前元素t[j-1]删除, 那么dp[i][j]的数值要看s[i-1]与t[j-2]的比较结果了, 即dp[i][j] = dp[i][j-1]

115. 不同的子序列

题目:

给你两个字符串 s 和 t ，统计并返回在 s 的 子序列 中 t 出现的个数，结果需要对 10^9 + 7 取模。

方法:

动态规划

  • dp数组以及下标含义:

    • dp[i][j]表示以下标j-1结尾的字符串t在以下标i-1为结尾的字符串s中出现的次数

  • 递推公式

    考虑以下两种情况:

    • s[i-1] == t[j-1], t中找到一个字符也在s中出现

      • 考虑s[i-1]这个元素, 则当前长度的串出现的次数等于当前长度-1的串出现的次数: dp[i][j] = dp[i-1][j-1]

      • 不考虑s[i-1]这个元素, 也就是之前的串就可能已经包含了t, 则dp[i][j] = dp[i-1][j]

      因此, 当前长度串在s中出现的次数等于以上两种情况之和, 即: dp[i][j] = dp[i-1][j-1] + dp[i-1][j]

    • s[i-1] != t[j-1], 相当于s要删除字符再匹配, s如果把当前元素s[j-1]删除, 那么dp[i][j]的数值要看s[i-2]与t[j-1]的比较结果了, 即dp[i][j] = dp[i-1][j]

  • 初始化

    当t长度为0时, 其在s中的次数为1, 因此dp[i][0]要初始化为1;

    同理dp[0][j]要初始化为0, dp[0][0]为1
