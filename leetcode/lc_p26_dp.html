53.最大子序和

题目:

给你一个整数数组 nums ,请你找出一个具有最大和的连续子数组(子数组最少包含一个元素),返回其最大和。

子数组是数组中的一个连续部分。

方法:动态规划

dp数组以及下标含义:

dp[i] : 以nums[i]结尾的最大连续子序列和为dp[i].

递推公式:

dp[i] 可由两个方向推得:

  • dp[i - 1] + nums[i] : nums[i] 加入当前连续子序列和

  • nums[i] : 从i开始计算当前连续子序列和

由dp数组定义可知, 需要一个临时变量记录结果值(即最大dp[i]的值)

392. 判断子序列

题目:

给定字符串 s 和 t ，判断 s 是否为 t 的子序列。

字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。(例如，"ace"是"abcde"的一个子序列，而"aec"不是)。

方法:

  1. 动态规划, 思路与1143.最长公共子序列差不多, 唯一的区别在递推公式.

  • dp数组以及下标含义:

    • dp[i][j]表示以下标i-1为结尾的字符串s, 和以下标j-1结尾的字符串t, 相同的子序列长度为dp[i][j]

  • 递推公式

    考虑以下两种情况:

    • s[i-1] == t[j-1], t中找到一个字符也在s中出现, dp[i][j] = dp[i-1][j-1] + 1

    • s[i-1] != t[j-1], 相当于t要删除字符再匹配, t如果把当前元素t[j-1]删除, 那么dp[i][j]的数值要看s[i-1]与t[j-2]的比较结果了, 即dp[i][j] = dp[i][j-1]

115. 不同的子序列

题目:

给你两个字符串 s 和 t ，统计并返回在 s 的 子序列 中 t 出现的个数，结果需要对 10^9 + 7 取模。

方法:

动态规划

  • dp数组以及下标含义:

    • dp[i][j]表示以下标j-1结尾的字符串t在以下标i-1为结尾的字符串s中出现的次数

  • 递推公式

    考虑以下两种情况:

    • s[i-1] == t[j-1], t中找到一个字符也在s中出现

      • 考虑s[i-1]这个元素, 则当前长度的串出现的次数等于当前长度-1的串出现的次数: dp[i][j] = dp[i-1][j-1]

      • 不考虑s[i-1]这个元素, 也就是之前的串就可能已经包含了t, 则dp[i][j] = dp[i-1][j]

      因此, 当前长度串在s中出现的次数等于以上两种情况之和, 即: dp[i][j] = dp[i-1][j-1] + dp[i-1][j]

    • s[i-1] != t[j-1], 相当于s要删除字符再匹配, s如果把当前元素s[j-1]删除, 那么dp[i][j]的数值要看s[i-2]与t[j-1]的比较结果了, 即dp[i][j] = dp[i-1][j]

  • 初始化

    当t长度为0时, 其在s中的次数为1, 因此dp[i][0]要初始化为1;

    同理dp[0][j]要初始化为0, dp[0][0]为1

583. 两个字符串的删除操作

题目:

给定两个单词 word1 和 word2 ，返回使得 word1 和 word2 相同所需的最小步数。

每步 可以删除任意一个字符串中的一个字符。

动态规划一

乍一看感觉就是求最长公共子序列, 然后用字符串长度相减得到需要删除的字符数.
一顿操作然后提交, 发现直接通过了😂

动态规划二

如果和115.不同的子序列相比, 则是两个字符串都可以删除字符了, 情况稍微复杂一些, 思路不变.

  • dp数组及下标含义:

    以i-1结尾的 word1, 和以j-1结尾的 word2, 想要相等所需要删除的最少次数为dp[i][j].

  • 递推公式

  • 当word1[i-1] == word2[j-1]时, 不需要删除字符, dp的值由 dp[i-2][j-2] 决定:

    dp[i][j] = dp[i-1][j-1]

  • 当word1[i-1] != word2[j-1]时, 有三种情况:

    1. 删 word1[i-1], 增加一次删除数, dp[i][j] = dp[i-1][j] + 1

    2. 删 word2[j-1], 增加一次删除数, dp[i][j] = dp[i][j-1] + 1

    3. 同时删 word1[i-1] 和 word2[j-1], 增加两次删除数, dp[i][j] = dp[i-1][j-1] + 2

    最后取最小值: dp[i][j] = min({dp[i-1][j] + 1, dp[i][j-1] + 1, dp[i-1][j-1] + 2})

    又因为 dp[i][j-1] + 1 = dp[i-1][j-1] + 2, 所以可以简化为dp[i][j] = min(dp[i-1][j], dp[i][j-1])

    这里从字面上理解就是, 当同时删 word1[i - 1] 和 word2[j - 1], dp[i][j-1] 本来就不考虑 word2[j - 1]了，那么再删 word1[i - 1]，就达到两个元素都删除的效果，即 dp[i][j-1] + 1

    由递推公式可知, dp[i][0]和dp[0][j]一定需要初始化

  • 遍历顺序

    由递推公式可知, 遍历顺序为从上到下, 从左到右.

题目:

给你两个单词 word1 和 word2, 请返回将 word1 转换成 word2 所使用的最少操作数 .

  • 插入一个字符
  • 删除一个字符
  • 替换一个字符

示例1:

输入: word1 = “horse”, word2 = “ros”
输出: 3
解释:
horse -> rorse (将 ‘h’ 替换为 ‘r’)
rorse -> rose (删除 ‘r’)
rose -> ros (删除 ‘e’)

输入: word1 = “intention”, word2 = “execution”
输出: 5

647. 回文子串

题目:

给你一个字符串 s , 请你统计并返回这个字符串中 回文子串 的数目。

回文字符串 是正着读和倒过来读一样的字符串。

子字符串 是字符串中的由连续字符组成的一个序列。

1. 动态规划

采用中心扩散来判断回文串

dp[i][j]: [i,j]的s是否是回文串

递推公式考虑如下几种情况:

  • s[i] == s[j]:
  1. i == j, 一个字符必定是回文串;
  2. i+1 == j, i和j相邻且s[i]==s[j], 此时也是回文串, 如: aa;
  3. j-i > 1, 取决于dp[i+1][j-1], 如: axxxxxxa, 取决于中间的串是否是回文串.

遍历顺序, 由 dp[i][j] = dp[i+1][j-1] 可知, 从下往上, 从左往右遍历.
