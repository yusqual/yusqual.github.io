739.每日温度

题目:

给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 0 来代替。

单调栈

单调栈: 设置一个栈保存遍历过的元素的下标, 保证栈中下标对应的元素值单调递增或递减
递增则求某个元素”右边”第一个比他大的元素

顺序遍历每个元素, 如果栈为空或者当前元素比栈顶元素小, 则将当前元素入栈;
如果当前元素比栈顶元素大, 则 一直 对栈顶元素进行比较, 只要栈顶元素小, 就对栈顶元素记录对应的结果, 然后弹出, 继续对栈顶元素比较.

对于每个 0 <= i < nums1.length ，找出满足 nums1[i] == nums2[j] 的下标 j ，并且在 nums2 确定 nums2[j] 的 下一个更大元素 。如果不存在下一个更大元素，那么本次查询的答案是 -1 。

503. 下一个更大元素 II

题目:

给定一个循环数组 nums （ nums[nums.length - 1] 的下一个元素是 nums[0] ），返回 nums 中每个元素的 下一个更大元素 。

数字 x 的 下一个更大的元素 是按数组遍历顺序，这个数字之后的第一个比它更大的数，这意味着你应该循环地搜索它的下一个更大的数。如果不存在，则输出 -1 。

单调栈

题目说是循环数组, 实际只需要将nums数组复制一遍, 再进行单调栈的操作即可.
需要注意的是在记录结果时需要对nums.size()求余.

42. 接雨水

题目:

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图, 计算按此排列的柱子, 下雨之后能接多少雨水.

示例:
输入: height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出: 6
下面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图, 在这种情况下, 可以接 6 个单位的雨水（蓝色部分表示雨水）.

单调栈:

单调栈采用’行’来计算
遍历每个高度,寻找右边大的第一个元素, 此时会形成一个’凹槽’, 栈顶元素就是凹槽的底部, 下一个栈顶元素是凹槽的左部, 大的元素是凹槽的右部, 根据左右部最矮的那个高度减去底部元素高度得到雨水高度, 再根据左右部距离得到雨水宽度, 两者相乘就是雨水体积.
如果凹槽底部是第一个或者最后一个元素, 栈中没有下一个栈顶元素, 判断栈为空则不计算即可. (下图引用自 代码随想录)

84. 柱状图中最大的矩形

题目:

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

求在该柱状图中，能够勾勒出来的矩形的最大面积。

单调栈

和上一题接雨水相呼应, 接雨水求右边第一个大, 也就是找’凹槽’, 本题求右边第一个小, 找的是’凸起’;
对应的, 单调栈内元素应该从大到小(栈顶到栈底).
