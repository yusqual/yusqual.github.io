12. 事件钩子系统

HookEvent

注册游戏事件回调。

tParams 中注入的辅助字段（根据原始事件参数自动添加）：

  • tParams["_player"] → userid 对应的玩家
  • tParams["_victim"] → victim 对应的玩家
  • tParams["_attacker"] → attacker 对应的玩家
  • tParams["_subject"] → subject 对应的玩家
  • tParams["_entity"] → entityid 对应的实体

UnhookEvent

移除事件或特定回调。

UnhookAllEvents

移除所有注册的游戏事件。

AddChangeHook(Function) 添加变更钩子
RemoveChangeHook() 移除变更钩子

20.8 CTimer

封装一个延迟调用。

一般不直接使用，由 CreateTimer() 内部创建。

20.9 IScriptPlugin

脚本插件抽象接口。

所有方法默认抛出 PureVirtualError，子类必须重写。见 core-system.html §2。

21. 内部函数

以下函数由系统自动注册和调用，一般不需要直接调用：

函数 说明

LOU_OnRoundStart(tParams) round_start 事件：触发 g_Hooks.OnRoundStart，加载插件
LOU_OnRoundEnd(tParams) round_end 事件：触发插件卸载
LOU_OnPlayerSay(tParams) player_say 事件：分发聊天命令
AnalyzeCommand(objCommand, sArgs, hPlayer, sSeparator) 解析并分发聊天/用户命令
ButtonsListener_Think() 每帧检测所有玩家的按钮状态
CheckButtons(hPlayer, button) 检查单个玩家的单个按钮
InjectAdditionalClassMethods_Think() 一次性 tick：注入 CTerrorPlayer 方法后移除自身
UserConsoleCommand(hPlayer, sArgs) 分发用户控制台命令（仅当 g_bOverrideGameHooks == true）
